## 坐标系

地心惯性坐标系（i系）

* 定义：原点在地心，z轴指向北极，x轴指向春分点
* 特点：不随地球转动，牛顿定律严格成立

地球坐标系（e系）

* 定义：原点在地心，固连于地球
* 特点：与i系的区别在于，e系会随地球自转

导航坐标系（n系）

* 定义：原点在载体位置，三轴指向东北天（ENU）或北东地（NED）
* 特点：随载体移动，是导航解算的主要参考系

载体坐标系（b系）

* 定义：固连于载体，三轴与载体结构对齐
* 特点：传感器测量值的原始输出坐标系

坐标系转换关系

* 方向余弦矩阵
* 欧拉角
* 四元数
* 链式法则
